# Tables/Comandas - Quick Start Guide

## For Store Owners/Managers

### Setting Up Tables

1. **Check Your Plan**
   - Starter: No tables (upgrade to Pro or Premium)
   - Pro: Up to 10 tables
   - Premium: Up to 30 tables

2. **Create Your First Table**
   - Go to "Mesas" in the sidebar
   - Click "Nova Mesa"
   - Enter table number (e.g., 1, 2, 3...)
   - (Optional) Add name/label (e.g., "Varanda", "Salão")
   - Enter capacity (e.g., 4 seats)
   - Click "Criar Mesa"

3. **Create All Your Tables**
   - Repeat for each physical table in your store
   - Use logical numbering (1, 2, 3... or 101, 102, 103...)
   - Stay within your plan limit

## For Attendants

### Opening a Tab (Comanda)

1. **Navigate to Tables**
   - Click "Mesas" in sidebar
   - See all tables with current status

2. **Select a Free Table**
   - Tables in white = free
   - Tables in green = occupied
   - Click on a free table

3. **Open the Tab**
   - (Optional) Enter customer name
   - Click "Abrir Comanda"
   - Table is now marked as occupied

### Adding Items to a Tab

1. **Open the Tab**
   - Click on occupied table
   - See current items and total

2. **Add Item**
   - Click "Adicionar Item"
   - Select product from dropdown
   - Enter quantity
   - (Optional) Add notes (e.g., "sem açúcar")
   - Click "Adicionar"

3. **View Running Total**
   - Total updates automatically
   - Shows all items with quantities and prices

### Removing Items

1. **Find the Item**
   - In the tab view, see all items
   - Each item has a trash icon

2. **Delete Item**
   - Click trash icon next to item
   - Confirm deletion
   - Total updates automatically

### Closing a Tab

1. **Review the Tab**
   - Check all items are correct
   - Note the total amount

2. **Close Tab**
   - Click "Fechar Comanda" (red button at top)
   - Confirm action
   - **Important:** Tab closes but no financial transaction created yet
   - Table becomes free

## Color Guide

### Table Status Colors

**White Background**
- Status: Free
- Meaning: No active tab, available for new orders
- Action: Click to open new tab

**Green Background**
- Status: Occupied
- Meaning: Has active tab with orders
- Details: Shows customer name (if entered) and running total
- Action: Click to view/edit tab

**Gray Background**
- Status: Inactive
- Meaning: Table disabled/not in use
- Action: Cannot be used (admin must activate)

## Common Workflows

### Scenario 1: Customer Arrives

1. Customer arrives at table 5
2. Go to Mesas
3. Click on "Mesa 5" (should be white/free)
4. Enter customer name (optional)
5. Click "Abrir Comanda"

### Scenario 2: Taking Order

1. Tab is open for table 5
2. Click on "Mesa 5" (now green/occupied)
3. Click "Adicionar Item"
4. Select "Açaí 500ml"
5. Quantity: 1
6. Notes: "com morango"
7. Click "Adicionar"
8. Repeat for each item

### Scenario 3: Customer Wants to Add More Items

1. Click on occupied table
2. Click "Adicionar Item"
3. Add new items
4. Total updates automatically

### Scenario 4: Customer Made a Mistake

1. Click on occupied table
2. Find incorrect item
3. Click trash icon next to item
4. Confirm deletion
5. Add correct item if needed

### Scenario 5: Customer Finished

1. Click on occupied table
2. Review all items
3. Note total amount
4. Click "Fechar Comanda"
5. Confirm closure
6. **Note:** Collect payment separately (tab closure doesn't process payment yet)

## Tips & Best Practices

### For Efficiency

1. **Use Table Numbers Logically**
   - Match physical table layout
   - Use sequential numbering
   - Add location in name (e.g., "Mesa 1 - Varanda")

2. **Add Customer Names**
   - Helps identify tabs quickly
   - Useful when multiple attendants working
   - Can be added/edited anytime

3. **Use Notes Field**
   - Special requests (e.g., "sem leite condensado")
   - Dietary restrictions
   - Custom preparations

### Avoiding Mistakes

1. **Check Table Status**
   - Always verify table is free before opening new tab
   - System prevents duplicate tabs, but double-check

2. **Review Before Closing**
   - Check all items are correct
   - Confirm quantities
   - Verify total with customer

3. **Don't Close Tab Until Customer Pays**
   - Tab closure is final
   - Cannot reopen closed tab
   - Make sure payment is collected first

## Plan Upgrade Guide

### When to Upgrade

**From Starter to Pro:**
- You want to use tables/comandas feature
- You have up to 10 tables
- Cost: Check "Minha Assinatura"

**From Pro to Premium:**
- You need more than 10 tables
- You have up to 30 tables
- Cost: Check "Minha Assinatura"

### How to Upgrade

1. Click "Minha Assinatura" in sidebar
2. View available plans
3. Click upgrade button
4. Follow payment instructions
5. Tables feature activates immediately after payment

## Troubleshooting

### "Recurso Bloqueado" Message
- **Problem:** You're on Starter plan
- **Solution:** Upgrade to Pro or Premium

### "Limite de mesas atingido"
- **Problem:** You've reached your plan's table limit
- **Solution:** Upgrade to higher plan or deactivate unused tables

### "Mesa já possui comanda aberta"
- **Problem:** Trying to open second tab on occupied table
- **Solution:** Close existing tab first or use different table

### Table Stuck as "Occupied" with No Tab
- **Problem:** Tab was closed but table status not updated
- **Solution:** Refresh page, contact admin if persists

### Cannot Add Items to Tab
- **Problem:** No active products in store
- **Solution:** Admin/Manager must activate products first
